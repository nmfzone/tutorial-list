### Retrive input from STDIN
##### The convention
Symbol | Description
------ | -----------
%d | int (signed decimal integer)
%u | unsigned decimal integer
%f | floating point values (fixed notation) - float, double
%e | floating point values (exponential notation)
%s | string
%c | character
