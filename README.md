###Laravel :
```
  location / {
        try_files $uri $uri/ /index.php?$query_string;
  }
```

###Codeigniter :
```
  location / {
        try_files $uri $uri/ @ci_index;
  }
  
  location @ci_index{
	rewrite ^(.*) /index.php?$1 last;
  }
```
